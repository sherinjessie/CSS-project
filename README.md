# CSS-project

A simple perfume landing page built using **HTML** and **CSS**.

## ✨ About

Perfumy is a beginner-friendly front-end project that focuses on creating a clean and attractive user interface using only HTML and CSS.

## 🚀 Features

- 🌸 Simple and elegant design
- 📱 Responsive layout
- 🎨 Clean user interface
- ⚡ Easy to navigate

## 🛠️ Built With

- HTML5
- CSS3

## 📁 Project Structure

```
Perfumy/
├── index.html
├── style.css
└── README.md
```

## 💻 Run Locally

1. Download or clone this repository.
2. Open `index.html` in your browser.
